\section{Research Interests}

I'm interested in verifying concurrent storage systems, especially file systems.
My research uses formal verification to rule out bugs by proving that the
system's implementation always follows a specification of its intended behavior.
Storage is especially interesting because of the importance of handling \emph{crashes}
(the system should retain your data even if the computer suddenly turns off) and
\emph{concurrency} (for performance, the system should use multiple CPU cores).
